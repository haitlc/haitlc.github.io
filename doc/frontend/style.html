What is CSS3?

CSS3 is the latest standard for CSS, completely backwards-compatible with earlier versions. Modern browser and UI framework make use of it.

Some important new features:

  • Selectors
  • Box Model
  • Backgrounds and Borders
  • Image Values and Replaced Content
  • Text Effects
  • 2D/3D Transformations
  • Animations
  • Multiple Column Layout

What are difference between SASS, SCSS, LESS?

They are CSS preprocessors, providing programming features like function,variable,mixins,etc, finally compile into lengthy CSS which is used by browser.
Actually their differences are very small, all of them can provide most features.

SASS SCSS LESS
History Built on 2007. Coded in Ruby
Extend from SASS Built on 2009. Coded in JavaScript
Syntax Indented syntax, like Ruby and Python >CSS-like syntax.
Use brace{} to denote code block and semicolons; to separate lines		 CSS-like syntax.
Feature First to support selector inheritance
High learning curve to setup Ruby and syntax
Compile to css by Ruby		 Same as SASS but low learning curve in syntax Variable, nesting, mixin, functions and selector inheritance
Low learning curve
Real-time compilation by JavaScript (include less.js after .less)

Example

SASS SCSS
Variable 
$color: #FFF
body 
  color: $color
 
$color: #FFF;
body {
  color: $color;
}
Nesting 
nav
  ul
    margin: 0
    padding: 0
  li
    display: inline-block
    a
      display: block
      text-decoration: none
 
nav {
  ul {
    margin: 0;
    padding: 0;
  }
  li { 
    display: inline-block; 
    a {
      display: block;
      text-decoration: none;
    }
  }
}
Mixin 
=error($borderWidth: 2px, $color: #F00)
  border: $borderWidth solid $color
  color: $color

.default-error
  +error()

.custom-error
  +error(5px, #F11)
 
@mixin error($borderWidth: 2px, $color: #F00) {
  border: $borderWidth solid $color;
  color: $color;
}
.default-error{
  @include error()
}
.custom-error{
  @include error(5px, #F11)
}
Inheritance 
.message
  border: 1px solid #ccc
  padding: 10px
  color: #333

.success
  @extend .message
  border-color: green

.error
  @extend .message
  border-color: red

.warning
  @extend .message
  border-color: yellow 
.message {
  border: 1px solid #ccc;
  padding: 10px;
  color: #333;
}
.success {
  @extend .message;
  border-color: green;
}
.error {
  @extend .message;
  border-color: red;
}
.warning {
  @extend .message;
  border-color: yellow;
}
